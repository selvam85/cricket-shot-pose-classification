Cricket Shot Classification using TensorFlow.js BlazePose model

This demo classifies between three different cricket shots - cover drive, flick and square cut. BlazePose model was used to capture the poses from images belonging to the above category of shots and then the pose data was normalized and passed as input to a KNN classifier for classification.
